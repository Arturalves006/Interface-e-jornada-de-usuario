# Interface-e-jornada-de-usuario
#### 1° Passo (Criação de um wireframe mobile)
 Este projeto desenvolvido no Figma consiste na criação de um aplicativo mobile focado em facilitar o agendamento de consultas de forma rápida e intuitiva. O objetivo    principal é oferecer ao usuário uma experiência simples, com navegação clara e dois caminhos distintos para realizar a mesma ação: marcar uma consulta.
